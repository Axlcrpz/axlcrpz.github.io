# Image assets

Drop project screenshots, your photo, and the OG/social preview image here.

## How to use a project screenshot
1. Save the image in this folder, e.g. `edu-system.png`.
2. Open `js/data/projects.js`.
3. Set the project's `image` field:
   ```js
   image: "./assets/img/edu-system.png",
   ```
The card will use your screenshot instead of the gradient placeholder.

Recommended: ~1280×800px, optimized (`.webp` or compressed `.png`/`.jpg`).
